# Mally Wear App

Proyecto de la app Mally Wear para Android.

## Cómo usar

1. Instalar Python y NodeJS en Termux.
2. Ejecutar `python server.py` para probar la web local.
3. Ejecutar `python builder.py` para generar APK listo en Descargas.
4. Ejecutar `./make_git.sh` para crear y subir el proyecto a GitHub automáticamente.
